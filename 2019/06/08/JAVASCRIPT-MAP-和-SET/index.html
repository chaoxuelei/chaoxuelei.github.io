    Map

    怎么用呢？

    一个key只能对应一个value，所以，多次对一个key放入value，后面的值会把前面的值冲掉，就像这样：

    Set

    Set和Map类似，也是一组key的集合，但不存储value。由于key不能重复，所以，在Set中，没有重复的key。
